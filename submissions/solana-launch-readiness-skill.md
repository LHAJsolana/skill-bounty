# solana-launch-readiness-skill

Repository: https://github.com/LHAJsolana/solana-launch-readiness-skill

## Summary

solana-launch-readiness-skill is a production-ready Solana AI Kit skill for reviewing Solana projects before mainnet launch.

It helps AI coding agents produce structured launch readiness reports covering security triage, transaction reliability, RPC resilience, wallet UX, monitoring, incident response, token launch readiness, documentation, support, and founder/GTM preparation.

## What It Includes

The repo includes a progressive SKILL.md, focused skill modules, optional agents, command workflows, safety rules, demo outputs, reference files, JSON schema, tests, GitHub Actions CI, safe installer, README, CONTRIBUTING guide, CHANGELOG, and MIT license.
